# 前言

密码学在保护关键系统和敏感信息方面有着悠久而重要的历史。本书将向您展示如何使用 Python 加密、评估、比较和攻击数据。总的来说，本书将帮助您处理加密中的常见错误，并向您展示如何利用这些错误。

# 这本书是给谁的

本书面向希望学习如何加密数据、评估和比较加密方法以及如何攻击它们的安全专业人员。
